The LWUIT Browser demo application is a simple mobile browser built on top of HTMLComponent, which is a component that knows how to render HTML documents the conform to the XHTML-MP 1.0 standard.

The application includes several local html files (in the jar) that can be accessed from the main page. These files demonstrate various features of HTMLComponent. In addition it is possible to browse to web sites that conform to the XHTML-MP 1.0 standard. While there are not many sites that conform 100% to that, a lot of mobile sites can be rendered in this browser, as parsing is set here to be less strict.

Toolbar - On the top you can see a nacigation toolbar and like any other browser you will find there an address bar in which you can simply type the address you wish to browse to, and navigation buttons: Back, Cancel, Forward, Refresh and Home.

Menu - On the menu you can find all of the navigation commands, and also a command called images that toggles image downloading.

Shortcuts - There are several keypad shortcuts that can ease up navigation, note that these are active only in case the focus is not a text field or area:
1 - Top 2 - None 3 - Page Up
4 - Back 5 - Refresh 6 - Forward
7 - End 8 - None 9 - Page Down
* - Navbar 0 - Stop # - Home page
